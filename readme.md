# 作ったもののディスクリプション・使い方
ある曲の歌詞を感情分析し、その曲をブラウザで流せます。
まず「感情分析APIいまにゅ.ipynb」内のtext=""に、分析したい歌詞を入力して、「セルの実行」を行ってください。
次に「WebAPI.html」内のvideoId: '',に、見たいYouTube曲のIDを入力してください。ブラウザを開くと自動的に動画が流れます。
